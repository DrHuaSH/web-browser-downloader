# Web浏览器下载器

一个基于Web的浏览器应用，可以部署在GitHub Pages等静态托管平台上。该应用提供内置浏览器功能，并具备智能内容检测和下载能力，包括音频视频文件、文本内容和PDF文档的自动识别与下载。

## 功能特性

- 🌐 **完整浏览器界面** - 包含地址栏和浏览区域的完整浏览器体验
- 🎵 **音视频下载** - 自动检测和下载网页中的媒体文件
- 📝 **文本提取** - 提取网页内容并转换为Markdown格式下载
- 📄 **PDF下载** - 检测和下载网页中的PDF文档
- 🔒 **安全传输** - 强制使用HTTPS协议，保护数据传输安全
- 📱 **移动适配** - 完美支持手机和平板设备
- 🚀 **静态部署** - 可部署在GitHub Pages等静态托管平台

## 移动设备支持

### iOS设备
- 文件会在新标签页中打开
- 用户可通过分享功能保存内容
- 支持触摸友好的界面设计

### Android设备
- 支持标准的文件下载功能
- 文件直接保存到下载文件夹
- 完整的下载进度显示

## 技术架构

- **前端**: 原生JavaScript + HTML5 + CSS3
- **响应式设计**: CSS Grid + Flexbox + Media Queries
- **安全性**: HTTPS强制使用 + CORS代理
- **移动适配**: 触摸优化 + 设备检测

## 部署说明

### GitHub Pages部署

1. Fork或下载此项目
2. 在GitHub仓库设置中启用GitHub Pages
3. 选择主分支作为源
4. 访问 `https://yourusername.github.io/repository-name`

### 本地开发

由于CORS限制，建议使用本地服务器运行：

```bash
# 使用Python
python -m http.server 8000

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 使用说明

1. **浏览网页**: 在地址栏输入网址，点击"访问"按钮
2. **下载媒体**: 当检测到音视频文件时，点击"音视频下载"按钮
3. **提取文本**: 点击"文本下载"按钮提取并下载页面文本内容
4. **下载PDF**: 当检测到PDF文件时，点击"PDF下载"按钮

## 安全特性

- 所有网络通信强制使用HTTPS加密
- 通过CORS代理隐藏用户真实IP地址
- 敏感数据在本地处理，不存储在服务器
- 自动清理潜在的敏感信息
